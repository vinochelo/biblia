# **App Name**: Explorador de Versículos

## Core Features:

- Configuración de Clave API: Configura de forma segura la clave API de API.Bible para acceder a la API de la Biblia.
- Búsqueda de Versículos: Permite a los usuarios buscar versículos o palabras clave específicas en múltiples versiones de la Biblia.
- Gestión de Versiones: Muestra múltiples versiones de la Biblia dentro de la aplicación, asegurando la compatibilidad con el formato API unificado.
- Monitoreo de Uso: Realiza un seguimiento del uso de la API para mantenerse dentro de los límites del plan (5,000 para Starter, 150,000 para Pro) y evitar cargos por excedente. Utiliza el plan de uso seleccionado (comercial o no comercial) y configura la protección contra sobreuso.

## Style Guidelines:

- Color primario: Azul profundo (#3F51B5) para transmitir una sensación de reverencia y estabilidad.
- Color de fondo: Gris claro (#ECEFF1), una saturación muy baja del tono primario, proporciona un fondo limpio y evita distracciones del texto.
- Color de acento: Indigo (#5C6BC0), análogo al tono primario pero ligeramente más claro y menos saturado, proporciona interés visual sin ser discordante.
- Texto del cuerpo y titulares: 'Alegreya', una fuente serif para transmitir una sensación elegante, intelectual y contemporánea.
- Nota: Actualmente solo se admiten fuentes de Google.
- Utiliza iconos simples y claros para búsqueda, selección de versión y ajustes, manteniendo una interfaz respetuosa y ordenada.
- Mantén un diseño limpio y organizado con un amplio espacio en blanco para mejorar la legibilidad y el enfoque en el texto bíblico.
